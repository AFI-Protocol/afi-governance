# AFI Governance Specs

This folder contains the canonical machine-readable schema for Proposal-as-Signal.

- [`universal_proposal_schema.json`](./universal_proposal_schema.json) – v1.0 schema for formal proposals
- Validation harness lives in [`../afi_proposal_schema_test/`](../afi_proposal_schema_test/)

All governance actions must comply with this schema to be valid under the AFI Protocol.
